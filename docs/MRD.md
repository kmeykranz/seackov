# MRD - Underwater Extraction Treasure Run

## Audience
Players who enjoy short-session action roguelite loops and stealth-risk decisions.

## Competitive References
- Brotato-like arena readability: top-down movement, fast recognition, dense moment-to-moment decisions.
- Extraction games: carried value is unsafe until extracted.

## Marketable Hook
A flooded Earth recovery dive where every extra pickup visibly fills the backpack and increases pressure because a single detection clears the current-run haul before it reaches the boat.

## Prototype Positioning
This build validates the single-run loop, a minimal boat-side item handling loop, and the first persistent exploration unlock before adding full campaign flow, broader save data, procedural generation, or later story areas.
